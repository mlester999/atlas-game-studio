# SolTower Starlight Vault Assets

These PNGs are original deterministic local SolTower Starlight Vault assets generated by `scripts/generate-starlight-vault-assets.mjs`.

- Banner art: 960x320 RGBA PNGs with text-free fantasy summon compositions.
- Reward icons: 160x160 RGBA PNGs grouped by weapons, armor, relics, and costumes.
- Rarity frames: 160x160 transparent RGBA PNGs for Common, Uncommon, Rare, Epic, Legendary, and Mythical tiers.
- Utility icons: 64x64 transparent RGBA PNGs for gold, pity, rate-up, and featured states.

These files are committed project assets. Do not replace them with external URLs, emoji, CSS-only shapes, or missing manifest entries.
